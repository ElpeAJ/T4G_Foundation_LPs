  • Lesson 8: Overview

    This lesson covers Cascading Stylesheet - the various methods of styling (inline, embedded and external), CSS syntax, property-value pairs and CSS selectors.

    Tasks

    1. 1. Learn how to use inline CSS. Use style="" as an attribute inside the element
    2. 2. Learn how to use embedded CSS. Use style tags / elements inside the head element
    3. 3. Learn the syntax of Embedded CSS and external CSS
      • i. CSS selectors
      • ii. curly braces
      • iii. property-value pairs
      • iv. terminator ;
    4. 4. Learn about Typography, the art of arranging text on a page. Use these properties:
      • i. font-family
      • ii. font-size
      • iii. font-weight
      • iv. font-style
      • v. text-transform
      • vi. letter-spacing(px)
      • vii. line-height
      • viii.text-align
    5. 5. Linking to an external CSS file.
    6. 6. Learn about CSS selectors - classes and ids.
    7. 7. Learn about Visual Rules. Use these properties:
      • i. color (foreground)
      • ii. background-color
      • iii. opacity (fully visible:0 - 1:fully visible or opaque)
      • iv. background-image
    8. 8. Learn about Box model. Use these properties:
      • i. width
      • ii. height
      • iii. border (width | style | color)
      • iv. border radius (% or px )
      • v. padding & padding shorthand(4 | 3 | 2 values)
      • vi. margin & margin shorthand(4 | 3 | 2 values)
      • vii. overflow
      • viii.Resetting Defaults
    9. 9. Use Devtools to inspect the webpage.
